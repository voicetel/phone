2. Verify checksum (optional)

Validate file integrity by comparing SHA-256 checksums.

  1. Open Windows PowerShell.
  2. Run: Get-FileHash .\VoiceTelPhone-Setup.exe -Algorithm SHA256
  3. Compare with the value published on your release page.

3. Install

  1. Double-click the downloaded .exe or .msi file.
  2. If Windows SmartScreen appears:
    1. Click More info.
    2. Click Run anyway.
  3. Follow the installation wizard to complete setup.

4. First launch

  1. Open VoiceTel Phone from the Start Menu.
  2. Allow microphone access when prompted.
    1. Go to Settings → Privacy & security → Microphone.
    2. Enable Microphone access and allow VoiceTel Phone.
  3. Enter your SIP credentials (domain, username, password) as provided by your admin.

Frequently Asked Questions

How do I uninstall or repair the app?

Open Settings → Apps → Installed apps. Locate VoiceTel Phone and choose Uninstall or Modify/Repair if available.

What if audio doesn’t work?

Check input/output devices in Windows Sound settings and verify microphone permissions are enabled.

Where can I find logs?

Logs are stored in %APPDATA%/VoiceTelPhone/logs. You can share them with support if needed.

Why does the installer request admin rights?

Admin privileges are required to register audio components and add VoiceTel Phone to the Start Menu.

Can I install without admin rights?

Yes. Use a portable ZIP build or request an MSI configured for per-user installation.
